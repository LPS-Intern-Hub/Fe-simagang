# Rekomendasi Menu Baru Dashboard Admin

Setelah menganalisis dashboard admin Anda saat ini, berikut adalah beberapa ide menu tambahan yang bisa membuat aplikasi Anda lebih lengkap dan profesional:

## 1. Pusat Pengumuman (Announcement Center)
- **Kegunaan**: Admin bisa membuat pengumuman yang langsung tampil di dashboard Intern dan Mentor.
- **Komponen**: Form input (Judul, Isi, Target Audience) dan riwayat pengumuman.

## 2. Laporan & Ekspor Data (Reporting)
- **Kegunaan**: Memungkinkan Admin mengunduh rekapitulasi data dalam format Excel atau PDF.
- **Data yang di-ekspor**: Daftar hadir bulanan, status logbook, dan penilaian akhir intern.

## 3. Manajemen Logbook Global
- **Kegunaan**: Admin bisa melihat semua logbook yang masuk dari seluruh intern (bukan hanya per intern di monitoring).
- **Fitur**: Filter berdasarkan tanggal, divisi, atau status (Pending/Approved).

## 4. Master Data Divisi & Jabatan
- **Kegunaan**: Mengelola daftar divisi (IT, HR, dll) secara terpusat agar tidak perlu diketik manual saat menambah user baru.

## 5. Log Aktivitas (Audit Log)
- **Kegunaan**: Mencatat aksi-aksi penting (seperti kapan admin menghapus user atau mengubah role) untuk transparansi.

---

### Rekomendasi Prioritas untuk Dikerjakan Sekarang:
Jika Anda ingin menambah fitur baru, saya menyarankan **Pusat Pengumuman** atau **Laporan Ekspor Data** karena ini yang paling sering dibutuhkan untuk operasional harian.
